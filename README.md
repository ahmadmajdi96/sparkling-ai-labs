# CortaNex AI

AI-powered business transformation platform. Simplifying Complexity.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
- GSAP Animations
- Three.js WebGL

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Production Build

```bash
npm run build
```

## Docker Deployment

```bash
docker-compose up --build
```

App will be available at `http://localhost:3001`
